# UAV-UGV 论文评分表

## 评分标准

| 维度 | 说明 | 范围 |
|------|------|------|
| **复现简易度** | 利用现有代码库复现的难易程度 | 0-100, 越高越简单 |
| **相关程度** | 与代码库相关性、算法可迁移程度 | 0-100, 越高越相关 |
| **本机复现** | R7 4800U + 16G RAM + 无显卡能否复现 | 100=能, 50=不太行, 0=不行 |
| **耗时评分** | 预计复现时间 | 1天=100, 3天=60, 1周=30, 其余=0 |

> **总分** = 复现简易度×0.3 + 相关程度×0.3 + 本机复现×0.2 + 耗时评分×0.2
